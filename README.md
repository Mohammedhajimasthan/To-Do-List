# ✅ To-Do List Application

A simple and responsive task management application that helps users organize daily activities, track progress, and improve productivity.

---

## 📌 Project Overview

The To-Do List Application allows users to create, manage, update, and delete tasks efficiently through a clean and user-friendly interface.

This project was developed to strengthen front-end development skills and demonstrate task management functionality using modern web technologies.

---

## ✨ Features

✅ Add New Tasks

✅ Edit Existing Tasks

✅ Mark Tasks as Completed

✅ Delete Tasks

✅ Responsive User Interface

✅ Real-Time Task Management

✅ Clean and Simple Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage

---

## 🚀 How It Works

1. Add a new task.
2. View all pending tasks.
3. Mark completed tasks.
4. Edit task details when required.
5. Delete completed or unwanted tasks.
6. Data remains available using browser storage.
